# batch_SRV-004
## [SRV-004]
- 불필요한 SMTP 서비스 실행 검사


## 취약점 검사 실험 및 실험환경
- OS : Windows Server 2016
- batch file
- service : SMTP

## 기준 
- 불필요한 SMTP서비스가 비활성화되어 있는 경우 양호 

## 배치파일 로직
- net start 로 불필요한 서비스 구동중인지 확인
- 확인 목록 IIS Admin, World Wide Web, Simple Mail Transfer Protocol, SMTP 등 ...
